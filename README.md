# littleCNN
Convolutional Neural Network  
for simple, easily.

## Usage
clone this repository
```
git clone https://github.com/satoshishimada/littleCNN.git
cd littleCNN
```
### Samples
* logic  
compile sample
```
make logic
```

* mnist  
download dataset
```
cd dataset/mnist
./get_mnist.sh
```
compile sample
```
make mnist
```
